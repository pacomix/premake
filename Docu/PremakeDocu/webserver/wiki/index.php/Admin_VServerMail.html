Login

Following is the information necessary to log in to the external mail server. Use the key-based login if possible.

  • Host name: h1653597.stratoserver.net
  • IP Address: 85.214.90.77
  • SSH running on port 42322
  • Root's private key: External Mailserver\root_ssh_key\private.ppk
  • Private key passphrase: look up External Mailserver in the password file
  • Root Password: look up External Mailserver in the password file

File Transfer

To transfer files to and from the server, you can use WinSCP with the SCP protocol. The credentials and the port to use are the same as for the SSH login. Using pageant, you can perform a key-based login.

Cron Jobs

A user-defined cronjob is used to send out mailserver stats each day. You can find its crontab entry in /etc/crontab. The e-mail address to send the stats to is in the script itself.

Postfix configuration

The postfix configuration can be found in /etc/postfix. Please refer to the comments in main.cf, transport and virtual for hints about how things are set up.

Backing Up Configuration and Scripts

To back up the postfix configuration and the scripts in /root/bin, perform the following steps:

  • Open an SSH shell connection to the server
  • Run the script /root/bin/backup.sh
  • Close the SSH shell, if no longer needed
  • Connect to the server with WinSCP
  • Change to the directory /private-backup on the remote side
  • Copy the appropriate files (they should have the current date and time in the name) to the local side
  • Save them to the location
    \\Server\Users\Administrator\External Mailserver\config-backup.
    This is also where older backups can be found.
